Claude Code · built-in command reference

Claude Code commands

The commands built into Claude Code, typed with a leading / inside a session. Your own project and personal commands (skills) appear in the same / menu; commands from a connected MCP server show up as /mcp__server__name. Type / to see what is live in your session.

Session & context

/clearStart fresh with empty context. The old chat stays in /resume.
/resumeReopen a past session. (/continue)
/compactSummarise the chat to reclaim context.
/contextShow how full the context window is.
/rewindRoll code and chat back to a checkpoint.
/exportSave the conversation to a text file.

Plan, models & memory

/planEnter plan mode: propose before editing. (or Shift+Tab)
/modelSwitch model and set the default.
/effortSet reasoning effort, low through max.
/configOpen settings. (/settings)
/permissionsManage allow / ask / deny tool rules.
/memoryEdit CLAUDE.md and auto-memory.
/initGenerate a CLAUDE.md for the project.

Code, git & review

/diffInteractive diff of uncommitted changes.
/code-reviewReview your working diff for bugs. (--fix, --comment)
/simplifyCleanup pass on the changed code.
/reviewReview a GitHub pull request.
/security-reviewScan pending changes for vulnerabilities.

Extend & connect

/agentsCreate and manage subagents.
/mcpConnect and manage MCP servers.
/hooksView hook configuration for tool events.
/runLaunch and drive your project's app.
/verifyBuild and run to confirm a change works.

Background, automation & remote

/backgroundDetach the session as a background agent. (/bg)
/tasksManage background work. (/bashes)
/loopRun a prompt on a repeating interval.
/goalKeep working until a stated condition is met. (/goal clear to stop)
/workflowsWatch and manage running workflows.
/remote-controlDrive this session from claude.ai. (/rc)

Account, help & health

/helpList the available commands.
/usageSession cost, plan limits, activity. (/cost)
/doctorDiagnose the installation and settings.
/feedbackReport a bug or share a session. (/bug)
/loginSign in. (/logout to sign out)
/exitQuit Claude Code. (/quit)

Keyboard shortcuts

Shift+TabCycle permission modes: plan, accept-edits, default.
EscInterrupt Claude, keeping work done so far.
Esc EscClear the input draft, or open rewind.
Ctrl+CInterrupt; press twice to exit.
Ctrl+OToggle the transcript viewer.
Ctrl+RSearch command history.
Ctrl+VPaste an image from the clipboard.
Up / DownStep through previous prompts.

Input & modes

/Open the command and skill picker.
!Shell mode: run a command, add its output to the session.
@Autocomplete a file path.
Shift+EnterNew line. Also Ctrl+J or \ then Enter.
Alt+TToggle extended thinking. (Option+T on macOS)
/btwAsk a side question without touching the conversation.
Availability varies by version, platform, and plan; type / to see what is live. Aliases shown in (parentheses).
Full docs: code.claude.com/docs/en/commands and /interactive-mode
